# client/src/pages/settings/OrderSettingsPage.jsx

## What is this file?

The **Order Settings page** at `/settings/orders`. A placeholder page for order configuration settings — currently no settings exist, so it displays an empty-state illustration with a message.

## What it shows

A centered icon, a heading, and a description ("No configurable order settings yet.") inside a bordered card. An Edit button is present but opens a modal that also shows a placeholder message.

## Relationship to other files

- Uses `OrderSettingsModal`: [components/modals/OrderSettingsModal.jsx](../../components/modals/OrderSettingsModal.md)
- See also: `ProductSettingsPage.jsx` and `YearInReviewSettingsPage.jsx` (identical layout)
